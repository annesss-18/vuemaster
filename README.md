VueMaster is a sophisticated Next.js web application designed to help users master job interviews through AI-driven simulations. By leveraging the power of Google's Gemini models, VueMaster offers realistic mock interviews tailored to specific job roles and technologies, providing actionable feedback to help candidates improve.

- **AI-Powered Interviewer:** Experience realistic technical interviews generated by Google's Gemini AI.
- **Customizable Sessions:** Choose your target role, tech stack, and experience level.
- **Real-time Feedback:** Receive comprehensive analysis of your answers, including ratings and improvement suggestions.
- **Secure Authentication:** User accounts managed securely via Firebase Authentication.
- **History Tracking:** Save and review past interview performance to track progress.
- **Modern UI/UX:** Built with Tailwind CSS and ShadCN UI for a responsive and accessible interface.

## ğŸ Getting Started

### Prerequisites

- **Node.js** (v20+)
- **npm** or **yarn**

### Environment Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/vuemaster.git
   cd vuemaster
   ```

2. Create a `.env.local` file in the root directory:
   ```bash
   cp .env.example .env.local
   ```
   *(Ensure you populate it with your specific API keys as shown below)*

3. Configure your environment variables:

   ```env
   # Firebase Client
   NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
   NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_project_id.firebaseapp.com
   NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
   NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_project_id.appspot.com
   NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
   NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id

   # Firebase Admin (Service Account)
   FIREBASE_PROJECT_ID=your_project_id
   FIREBASE_PRIVATE_KEY="-----BEGIN PRIVATE KEY-----\n..."
   FIREBASE_CLIENT_EMAIL=firebase-adminsdk-xxx@your_project_id.iam.gserviceaccount.com

   # Google AI
   GOOGLE_API_KEY=your_gemini_api_key
   ```

### Installation & Run

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to view the application.

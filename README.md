## ğŸ“¦ Getting Started

### Prerequisites
- Node.js 18+
- Firebase project with Auth and Firestore enabled
- Google AI Studio API key
- Upstash Redis instance (optional, for rate limiting)

### Installation

```bash
git clone https://github.com/your-username/intervoxai.git
cd intervoxai
npm install
```

### Environment Setup

Copy `.env.example` to `.env.local` and fill in your credentials:

```bash
cp .env.example .env.local
```

Required environment variables:
- `GOOGLE_GENERATIVE_AI_API_KEY` - Google AI Studio API key
- `NEXT_PUBLIC_FIREBASE_*` - Firebase configuration
- `FIREBASE_ADMIN_*` - Firebase Admin SDK credentials
- `UPSTASH_REDIS_*` - Upstash Redis credentials (optional)

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
